# Mitte
Mitte is a web-based token launchpad on Solana that allows users to create, manage tokens.

---
- Connect your wallet.
- Create a new SPL token by specifying name, symbol, and supply.
- Remove mint and freeze authorities for immutability.
---
### Steps:
```
npm install

// update .env
NEXT_PUBLIC_RPC_URL=

npm run dev

```
